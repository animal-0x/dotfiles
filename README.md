# Dotfiles Bootstrap

A minimal development environment setup focusing on:
- foot (Wayland) / alacritty (X11)
- helix
- zellij 
- nushell
- starship

## Quick Start
```bash
# 1. Clone the repo
git clone https://github.com/yourusername/dotfiles.git ~/dev/rice/dotfiles

# 2. Run setup script based on your distro
cd ~/dev/rice/dotfiles/setup/scripts
# For Arch/EndeavourOS:
./arch.sh
# For Debian/Ubuntu:
./debian.sh

# 3. Start using your new shell
nu
